## 이더넷
- tcp/udp 쓰고 싶으면 이더넷을 사용해야 합니다. 대략 50Mbps로 바이트스트림을 계속 뽑거나 먹여주고 싶으면 유일한 방법.

# GPIO출력
## HRPWM1A/B~HRPWM8A/B
- 빠른 PWM, 한 채널에 두 개씩 총 16개의 PWM 출력이 가능
- (Buffered 5V OUTPUT Fixed)
- GPIO 0~15핀
## EPWM9A/B~EPWM12A/B
- (Buffered 5V OUTPUT Fixed)
- GPIO 16~23핀
## 버퍼 입출력 조절용 핀
- GPIO 154 107 120 121 : OFF일 때 해당 버퍼 출력방향, ON일 때 해당 버퍼 입력방향, 런타임 중 값을 바꾸지 않기
    - 154: 64  65  66  67  68  69  70  71
    - 107: 80  81  82  83  84  85  86  87
    - 120: 88  89  90  91  92  93  94  133
    - 121: 127 128 147 95  101 102 103 104
## SCI통신
### SCIA
- RX GPIO 49(Non-Buffered 3.3V INPUT Fixed)
- TX GPIO 34(Non-Buffered 3.3V OUTPUT Fixed)
- easyDSP 통신용으로 고정돼있음
### SCIC
- RX GPIO 39(Buffered 5V INPUT Fixed)
- TX GPIO 38(Buffered 5V OUTPUT Fixed)
- easyDSP 통신용으로 고정돼있음

## SPI통신
### SPIB_SLAVE
- SPIB_CLK_IN GPIO 26(Buffered 5V INPUT Fixed)
- SPIB_SIMO GPIO 24(Buffered 5V INPUT Fixed)
- SPIB_SOMI GPIO 61(Buffered 5V OUTPUT Fixed)
- SPIB_STEn GPIO 27(Buffered 5V INPUT Fixed)

## UART
- 얘는 5V 버퍼도 해뒀지만, 3.3V도 열어뒀음. 둘 중에 한 곳에만 꽂기 바람
- UART_RX GPIO 43(Buffered 5V INPUT Fixed)
- UART_TX GPIO 42(Buffered 5V OUTPUT Fixed)
